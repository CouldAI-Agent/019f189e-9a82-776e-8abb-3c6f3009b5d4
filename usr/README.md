# 100 Portraits - Indian Names Gallery

A responsive Flutter application that displays a beautifully designed gallery of 100 distinct images, paired with 100 unique Indian names (50 male and 50 female). The application dynamically generates avatars for each person based on their name, ensuring every card is distinct.

## Features
- **100 Unique Names:** Carefully curated list of diverse Indian names.
- **Dynamic Images:** Uses an avatar generation API seeded by the person's name to ensure consistent, unique, and colorful profile images.
- **Responsive Layout:** The grid adapts automatically to phone, tablet, desktop, and web window sizes (from 2 columns on small screens up to 6 columns on ultra-wide screens).
- **Modern UI:** Uses Material 3 design, complete with gradient cards, soft drop shadows, loading indicators, and graceful error handling for images.

## Tech Stack
- Flutter
- Dart
- Material 3 

## Setup & Run
1. Ensure Flutter is installed and configured on your machine.
2. Clone or download this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run the app on your preferred device:
   ```bash
   flutter run
   ```
